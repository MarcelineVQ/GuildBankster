Guild Bankster
===

Bank-based restocks can be done using a mobile bank, but characters without a mobile bank can still bank restock by speaking with the following npcs:
* Horde banks:
* * Orgrimmar: Koma
* * Undercity: Randolph Montague
* * Thunder Bluff: Chesmu
* Alliance banks:
* * Darnassus: Any teller
